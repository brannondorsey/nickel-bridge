Ticket-stub counter: board/tournament progress drawn as a 1920s ADMIT ONE ticket; use in headers, lobby CTAs, and anywhere a count is the hero.
```jsx
<TicketStub label="BOARD" value="No. 2 of 4" width={164} />
```
Variants via props only — label/value/edgeText text; width scales the whole stub (compact board header ≈ 92).
